⚙️ Configuration Make.com requise

  • 1

    Dans le scénario existant, après le module Webhook, ajoute un module "Data stores — Add/Replace a Record".
    Champs : name → {{1.name}}  |  message → {{1.message}}  |  lang → {{1.lang}}  |  timestamp → {{1.timestamp}}  |  sessionId → {{1.sessionId}}

  • 2

    Crée un nouveau scénario Make.com avec un Custom Webhook comme déclencheur.

  • 3

    Ajoute un module "Data stores — Search Records" pour lire tous les enregistrements.

  • 4

    Ajoute un module "Webhooks — Webhook response" avec comme body :
    {"records": [{{toArray(3)}}]} (adapter selon Make.com)

  • 5

    Copie l'URL du nouveau webhook et colle-la dans MAKE_READ_WEBHOOK_URL ligne 14 de ce fichier. Change aussi ADMIN_PASSWORD.
